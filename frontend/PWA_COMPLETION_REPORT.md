## 🚀 Deployment Instructions

### Step 1: Build
```bash
cd frontend
npx.cmd vite build
```

### Step 2: Deploy
Choose one:

**Vercel** (Recommended)
```bash
npm i -g vercel
vercel --prod
```

**Netlify**
- Connect GitHub repo
- Build: `npm run build`
- Publish: `dist`

**Custom Server** (must be HTTPS)
- Copy `dist/` to server
- Configure HTTPS
- Set proper cache headers
